Casos de Prueba Automatizados
El archivo FormTests.cs incluye los siguientes escenarios:

Registro de Usuario Nuevo: Flujo completo que incluye generación de email dinámico, llenado de datos personales y validación de creación exitosa de cuenta.

Validación de Login: Verificación de acceso para usuarios existentes con manejo de redirecciones de URL.

Carrito de Compras: Selección aleatoria de productos, manejo de ventanas modales y validación matemática de los totales en el carrito.

Formulario de Contacto: Prueba de carga de archivos (PDF) y manejo de alertas de JavaScript (alert.Accept()).

Suscripción al Newsletter: Validación de suscripción exitosa en el pie de página (footer).
